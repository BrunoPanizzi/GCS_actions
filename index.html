Teste do github actions

Esse site está hospedado na AWS e é atualizado automaticamente pelo github actions!!

Esse é o código que faz o deploy:

name: Deploy na AWS
on:
push:
branches: master
jobs:
deploy:
runs-on: ubuntu-latest
name: Deploy
steps:
- name: Checkout
uses: actions/checkout@v2
- name: Set up AWS CLI
uses: aws-actions/configure-aws-credentials@v1
with:
aws-access-key-id: ${{ secrets.AWS_ACCESS_KEY_ID }}
aws-secret-access-key: ${{ secrets.AWS_SECRET_ACCESS_KEY }}
aws-region: sa-east-1
- name: Deploy to S3
run: aws s3 cp index.html s3://teste-gcs/index.html
